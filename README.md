# qB-Tracker-Assistant / qBittorrent Tracker 助手

2.0版本集成两个功能：

1.一键扫描失效种子，为它们打上标签。为避免误删，采取半自动方案，由用户自行处置；（不要随意删除种子，因为有些失效只是网站短期的问题）

2.批量替换tracker地址，当 PT 站更换域名（如从 .com 迁移至 .net）时，手动修改成百上千个种子的 Tracker 非常痛苦。本脚本为 qB 增加了一个快捷面板，支持“先检索、再排除、后替换”的严谨分步逻辑。

3.按分类转移种子到指定位置。

操作建议

在使用前，请把脚本代码开头@match一栏后面的地址自行修改为你qBittorrent的WEB_UI地址；
平时可以关闭这个脚本，毕竟大多数时候用不到它。

功能

安全第一：搜索替换tracker时强制校验 8 个字符，防止误删 Passkey。

去重逻辑：搜索替换自动识别已存在的 Tracker，避免一个种子出现重复地址。

高速并发：采用异步请求，处理几千个种子也只需几秒。

可视化：直接在 WebUI 右上角直接操作。
